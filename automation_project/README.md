# 🧪 V4MOS - Automação de Testes

Este repositório centraliza a estratégia e a execução dos testes automatizados para o projeto **V4MOS**. O objetivo principal é garantir a qualidade das entregas, prevenir regressões e acelerar o ciclo de feedback para o time de desenvolvimento.

---

## 🛠️ Stack Tecnológica

* **Framework principal:** Playwright
* **Linguagem:** JavaScript
* **Gerenciador de pacotes:** npm

---

## 🚀 Como Executar os Testes Localmente

### Pré-requisitos
Antes de começar, você precisará ter instalado em sua máquina:
* Node.js versão 18 ou superior
* Git

### Passo a Passo

1. **Clonar o repositório:**
   ```bash  
   git clone [https://github.com/seu-usuario/v4mos_test_automation.git](https://github.com/seu-usuario/v4mos_test_automation.git)
   cd seu-repo-de-testes

2. **Executar os testes em modo Headless (linha de comando):**
   ```bash
   npx playwright test

3. **Executar os testes com interface gráfica:**
   ```bash
   npx playwright test --ui
